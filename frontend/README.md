# Brand Narrative Forge

## Project Info

**URL**: Local Development

## How to run this code

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Start the development server**:
   ```sh
   npm run dev
   ```

3. **Build for production**:
   ```sh
   npm run build
   ```

## Technologies

- Vite
- React
- shadcn-ui
- Tailwind CSS
